🔴  Component Eval: Web Search Tool

Step 1 — Gold Standard

  • Create a list of gold standard web resources for test queries
  • These are the "correct" sources the search should return

Step 2 — Metric

  • Count how many results match gold standard
  • Calculate F1-score (precision + recall)

Step 3 — Tune Hyperparams

  • Search engine provider
  • Top-k results returned
  • Date range filter
  • Track metric as you vary each
