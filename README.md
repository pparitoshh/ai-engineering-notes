# AI Engineering Notes

A structured, open-source knowledge base for AI Engineering — built from reading, studying, and hands-on experimentation.

> **Goal**: A go-to reference for revisiting concepts across LLMs, evaluation, agentic AI, and more. Detailed enough to be useful, open enough to help others.

## How to Use

- Each topic folder has its own `README.md` with an overview and index.
- Notes are written to be **self-contained** — definitions, examples, and intuition in one place.
- Code snippets are in `/code` folders with comments explaining the concepts.
